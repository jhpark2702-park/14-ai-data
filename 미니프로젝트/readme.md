# 캘린더 만들기 프로젝트

calendar 모듈 없이 직접 달력을 시각화한 프로젝트입니다.
웹 브라우저에서 현재 월 기준 달력을 확인하고 이전달과 다음달로 이동할 수 있습니다.
또한 최근 근로자의 날이 공휴일로 지정되었기에 새로 추가했습니다.

---

## 파일 설명

- `calender_app.py`  
  Flask 서버 실행 파일이며, 달력 데이터를 생성합니다.

- `templates/calendar.html`  
  달력 화면을 출력하는 HTML 템플릿 파일입니다.

---

## 라이브러리 설치

실행 전 필요한 라이브러리를 설치해야 합니다.

```bash
pip install Flask
pip install holidays
```


---

## 실행 방법

터미널에서 프로젝트 폴더로 이동한 뒤 아래 명령어를 실행합니다.

```bash
python calender_app.py
```

실행 후 터미널에 아래와 비슷한 주소가 출력됩니다.

```text
http://127.0.0.1:5000/
```

이 주소를 브라우저에 입력하면 달력을 확인할 수 있습니다.
